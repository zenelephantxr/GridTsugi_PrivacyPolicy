## 概要（TL;DR）

- 本游戏的核心玩法可**离线运行**，**无需联网**即可游玩。
- 我们将**游戏数据本地保存在您的设备上**（例如分数、已完成关卡、用时，以及用于保留房间设置的空间锚点房间 ID）。
- 当您提交排行榜成绩时,您的 **PICO 用户 ID、显示名称（昵称）和分数**会通过 PICO 平台 SDK 传输至 PICO 平台服务器。
- 我们**不会**收集或展示您的 PICO 头像。
- 我们**不使用**任何第三方分析 SDK 或广告 SDK,**不会**出售或共享您的数据用于广告目的。
- GridTsugi：织桥迷宫包含在线排行榜功能,允许玩家与全球玩家及好友比较关卡完成时间,该功能由 PICO 平台提供支持。

## 我们处理的信息

### A. 游戏与进度数据（本地）

- **分数与表现指标**（例如得分、关卡完成时间）
- **进度跟踪**（例如已游玩关卡、已通过关卡、已完成关卡总数）
- **完成统计**（例如最佳时间、连胜、尝试次数）

我们仅将上述数据用于实现游戏核心功能（延续您的进度、显示统计与成就、设备端的难度调整）。

### B. PICO 平台用户数据（用于在线排行榜）

当在线排行榜功能启用时,我们通过 PICO 平台 SDK 收集并处理以下数据：

- **用户 ID（PICO ID / OpenID）** — 用于将您的成绩与您的账号关联。
- **显示名称（昵称）** — 显示在您的排行榜条目旁,以便玩家识别。
- **好友列表（仅限同样游玩本游戏的好友）** — 用于提供仅显示好友的排行榜视图。

除上述信息外,我们**不**收集或存储您的 PICO 头像、头像 URL、年龄段、邮箱或任何其他用户资料字段。

完整细节请参阅"在线排行榜功能"章节。

### C. 空间与房间设置数据（本地）

- **空间锚点房间 ID**（或类似的锚点／场景标识符）,用于记住您在房间中扫描／放置网格的位置,使您无需每次重新设置。

我们**不会**自行存储透视摄像头画面或原始房间扫描数据。所有房间映射、追踪和传感器处理均由 PICO 系统完成。我们仅保存一个本地标识符,供游戏定位您此前放置的内容。

### D. 您主动提供的联系信息

- 如果您**主动联系我们**（例如通过电子邮件或表单）,您可能会提供**姓名、邮箱地址或电话号码**。
- 此过程**在游戏之外**进行,仅用于回应您的咨询或客服请求。游戏运行**不需要**此类信息。

## 我们不收集或使用的信息

- 不收集任何头像、头像图片或任何形式的图像。
- 不存储面部追踪、摄像头帧或原始深度／网格数据。
- 不集成广告 SDK 或第三方分析 SDK。
- 不读取或存储生物识别或眼动追踪数据。
- 不出售或"共享"您的个人信息（按相关法律所定义）。

## 您的数据存储位置与传输

- **默认仅在设备本地：** 本政策所列的本地游戏数据保存在 PICO 设备上本游戏的私有存储空间内,**不会**传输给我们。
- **平台备份：** 如果您在系统层启用 **PICO 云备份**,您的设备可能会将本游戏的本地数据备份至 PICO 云端。该备份是由 PICO 与您的设备设置控制的平台功能 — 我们不控制也不接收该备份。
- **PICO 平台数据（排行榜）：** 用户 ID、昵称、好友列表（有限范围）以及排行榜分数会通过 PICO 平台 SDK 传输至并存储在 PICO 平台服务器。此处理由 PICO 受其自身隐私政策约束。
- **境内处理与存储：** 对于中国大陆 PICO 账号,相关数据由 **北京小鸟看看科技有限公司** 通过 PICO 的境内基础设施处理,并存储于位于中华人民共和国境内的服务器。

## 处理的法律依据

我们依据《中华人民共和国个人信息保护法》（以下简称"个人信息保护法"）处理您的个人信息,法律依据包括：

- **合同履行：** 为向您提供您选择游玩的游戏服务,处理本地游戏数据。
- **合法利益／服务必要：** 保存空间锚点 ID,使您无需重新扫描房间,提升可用性与游戏体验。
- **同意：** 在您登录 PICO 账号并提交排行榜成绩时,视为您同意将相关数据传输至 PICO 平台服务器（详见"在线排行榜功能"）。

我们不会将这些数据用于用户画像、广告投放或具有法律效力的自动化决策。

## 数据保存期限

- **游戏与空间锚点数据：** 仅保存在您的设备上,期限为您安装本游戏的整个期间,或直至您通过游戏内设置（如可用）或设备的应用数据管理功能清除／重置数据。
- 卸载本游戏通常会移除应用沙盒中存储的本地数据（具体取决于操作系统行为以及您在平台层启用的备份设置）。
- **PICO 平台排行榜数据：** 具体保留政策请参阅"在线排行榜功能"章节。

## 安全

- 数据存储在本游戏的**私有应用存储**中。我们依赖设备操作系统的沙盒机制与平台安全措施。
- 本游戏与 PICO 平台服务器之间的网络流量通过 PICO SDK 的安全通道（HTTPS／TLS）传输。
- 我们不在操作系统与 PICO SDK 提供的加密之外额外加密。由于本地数据默认完全保留在设备上（除非您启用系统备份）,第三方访问风险因此降低。

## 您的选择与控制

- **离线游玩：** 核心玩法无需联网。如果您不登录 PICO 账号即开始游玩,不会有任何排行榜数据被提交。
- **重置／清除数据：** 如果游戏提供"重置进度"或"清除锚点／房间设置"选项,您可以使用它来移除本地数据。您也可以卸载本游戏。
- **管理备份：** 在 PICO 设备设置中控制平台云备份。
- 如果您认为平台备份中包含您希望删除的游戏数据,请调整设备设置并按 PICO 的说明删除应用备份。
- **删除排行榜数据：** 通过下方邮箱联系我们,可申请删除您的排行榜条目及关联的 PICO 平台数据。

## 在线排行榜功能

GridTsugi：织桥迷宫包含由 PICO 平台 SDK 支持的在线排行榜功能,允许玩家与全球玩家及好友比较关卡完成时间。

### 排行榜收集的数据

当您登录 PICO 账号游玩本游戏时,我们通过 PICO 平台 SDK 收集并处理以下数据以提供排行榜功能：

- **用户 ID（PICO ID / OpenID）：** PICO 为您的账号分配的唯一标识。用于将分数与您的账号关联,并确定您在排行榜上的位置。
- **显示名称（昵称）：** 您的 PICO 显示名称。展示在排行榜上,使您和其他玩家能看到可识别的玩家身份,而非匿名 ID。
- **好友列表：** 同样游玩本游戏的 PICO 好友信息。用于提供仅显示好友的排行榜视图,并在全球排行榜上突出显示您的好友位置。我们只能看到同样安装本游戏并授予相关权限的好友。
- **游戏表现数据：** 您的关卡完成时间、分数、可选的并列分数指标,以及关卡／排行榜 API 名称标识。这些数据将提交至 PICO 排行榜服务器,用于计算和展示排名。

我们**不**收集头像、头像 URL、年龄段、邮箱或除用户 ID 与显示名称之外的任何资料字段。

### 我们如何使用这些数据

我们仅将上述数据用于以下目的：

- **提交并获取排行榜条目** — 记录您的完成时间并显示排名。
- **个性化排行榜显示** — 在排行榜上突出显示"您的位置"或"您的排名"。
- **展示玩家身份** — 在排行榜条目旁显示昵称以便识别。
- **启用好友相关功能** — 筛选排行榜以仅显示好友的成绩,或在全球排行榜上突出显示好友。
- **维护竞技公平性** — 将分数与特定用户账号关联,防止重复或欺诈性条目。

我们不将这些数据用于广告、用户画像、向第三方出售,或任何与提供排行榜功能无关的用途。

### 排行榜数据的存储位置

- 排行榜数据（用户 ID、分数、用时、昵称）通过 PICO 平台 SDK 传输至 PICO 平台服务器存储。我们**不运营独立的排行榜服务器**。
- 对于中国大陆账号,相关请求被路由至 PICO 境内基础设施,数据由北京小鸟看看科技有限公司在中国境内存储与处理。
- SDK 会根据已登录账号所在区域自动选择对应路由。

### 排行榜数据的保留

- **活跃排行榜条目：** 在 PICO 服务器上保留,直至超出所配置排行榜的范围或您申请删除为止。
- **用户资料数据（昵称）：** 仅用于显示目的临时缓存,并定期从 PICO 服务器刷新。
- **历史数据：** 可能由 PICO 平台服务依据其保留政策继续保存。

### 您的权利与选择

您对排行榜数据享有以下权利：

- **退出：** 您可以通过离线游玩或在启动本游戏时不登录 PICO 账号,选择不参与排行榜。本地进度依然有效。
- **请求删除：** 您可以通过下方邮箱联系我们以请求删除您的排行榜条目。我们将在 30 天内与 PICO 协作完成数据删除。
- **查看您的数据：** 您可以在游戏内排行榜界面查看您的条目。
- **控制资料可见性：** 您的昵称可见性可通过 PICO 账号设置进行管理。

## 您依据《个人信息保护法》享有的权利

依据《中华人民共和国个人信息保护法》,您对您的个人信息享有以下权利：

- **知情权与决定权：** 了解我们处理您个人信息的目的、方式与范围,并对处理活动作出决定。
- **查阅与复制权：** 查阅与复制我们处理的您的个人信息。
- **更正与补充权：** 要求更正或补充不准确或不完整的个人信息。
- **删除权：** 在法律规定的情形下要求删除您的个人信息。
- **撤回同意权：** 撤回您此前给予的处理同意（撤回不影响撤回前基于同意已进行的处理活动）。
- **解释说明权：** 要求我们对个人信息处理规则作出解释说明。
- **投诉与举报权：** 向相关监管部门投诉或举报。

如需行使上述权利,请通过下方"联系我们"中的方式与我们联系。我们将依照适用法律在合理期限内响应。

如果您对我们的处理活动存在任何疑虑,您也可以向国家网信部门或当地有关主管部门进行投诉或举报。
